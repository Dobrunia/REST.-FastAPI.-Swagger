### Рекомендации

1. REST подходит для публичных API, простой интеграции
2. gRPC оптимален для внутренних микросервисов с высокой нагрузкой
3. Оба протокола показывают стабильную работу без ошибок
4. Для production рекомендуется использование production-ready БД вместо SQLite
